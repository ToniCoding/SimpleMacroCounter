### v1.0.0 (Production planned: 2026-06-26) - SMC

**Dedicated branch**
`smc_1.0_stable`

**Release description**
The first stable release of SMC, delivering the core functionality initially planned for the software. Fully developed and tested.

**Features**
- **User system**: Registration, authentication, and session management.
- **Macro-nutrient management**: Manually add or reduce macro-nutrient intake.
- **Product registry**: Add new dishes to the product database.
- **Extensive product database**: Over 38,000 Spanish products integrated from OpenFoodFacts.
- **Configurable goals**: Set personalized macro-nutrient goals.
- **Piggy Bank**: Weekly calorie goal tracking with risk calculation for exceeding it.
- **History**: View past daily intakes.
- **Responsive UI**: Optimized for both mobile and desktop devices.
- **Optimized search**: Fast product search with pagination and market filters.

**Tech Stack**
- PHP (Symfony, Twig, Doctrine)
- MySQL
- JavaScript
- CSS
